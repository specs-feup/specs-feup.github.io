edge-distances

edge-distances: PropertyValueEdge<"intersection" | "node-position">

With value

  • "intersection" (default), the line from source to target
  • for "segment-weights" is from the outside of the source node’s shape to the outside of the target node’s shape.
  • With value "node-position", the line is from the source position to the target position. The "node-position" option makes calculating edge points easier — but it should be used carefully because you can create invalid points that intersection would have automatically corrected.

segment-distances

segment-distances: PropertyValueEdge<string | number | number[]>

A series of values that specify for each segment point the distance perpendicular to a line formed from source to target, e.g. -20 20 - 20.

  • Defined in lara-framework/Lara-JS/src-api/libs/cytoscape-3.26.0.d.ts:4323

segment-weights

segment-weights: PropertyValueEdge<string | number | number[]>

A series of values that weights segment points along a line from source to target, e.g. 0.25 0.5 0.75.A value usually ranges on [0, 1], with 0 towards the source node and 1 towards the target node — but larger or smaller values can also be used.
